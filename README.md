# Quant

Interactive Romanian guide to "How I Became a Quant" by Wall Street professionals.

## Overview

A web-based interactive guide in Romanian that explores the book "How I Became a Quant", featuring explanations and a markdown viewer. Provides a structured walkthrough of the book's content about quantitative finance careers.

## Contents

- `index.html` -- Main interactive guide
- `mdviewer.html` -- Markdown viewer
- `explicatii_quant_book.md` -- Romanian explanations
- `quant.txt` -- Source text

## Getting Started

```bash
bash run.sh
```

Or open `index.html` directly in a browser.
